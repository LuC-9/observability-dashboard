# Branding / Logo

Drop your logo here and the header will use it automatically.

- Put the file at: `src/assets/branding/logo.png` (or `.svg`).
- Recommended: transparent background, ~160×40 px (or square ~40×40).
- The header imports `logo.png`; if it's missing it falls back to the text wordmark
  "L'Oréal · GenAI Observability". To use a different filename, update the import in
  `src/components/AppHeader.tsx`.

Palette (see `src/theme.ts`): gold `#B6862C`, black `#0A0A0A`, ink `#1A1A1A`.
